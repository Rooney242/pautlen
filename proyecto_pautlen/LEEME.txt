Para ejecutar el compilador ejecute make dentro de los siguientes directorios:
1.	OO (hacer make pre si es la primera vez)
2.	PP/examen
El ejecutable estara dentro de PP/examen con el nombre "compilador"

El comando es: ./compilador <fichero_entrada> omicron.asm

El fichero del codigo asm sera omicron.asm

Una vez tenga el fichero asm haga "make asm" dentro de PP/examen para generar el otro ejecutable
